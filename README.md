# Python-Calculator
A menu-driven calculator built with Python

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Power
- √ Square Root
- 📜 View Calculation History
- 🗑️ Clear History
- ⚠️ Handles division by zero

## Concepts Used

- Variables
- Data Types (`int`, `float`, `str`, `list`)
- Functions
- Parameters & Return Values
- Loops (`while`, `for`)
- Conditional Statements (`if`, `elif`, `else`)
- Lists and List Methods
- User Input
- Type Conversion
- Exception Handling (`try`/`except`)
- f-Strings
- Menu-Driven Program Design
- Modular Programming


## How to Run

1. Make sure Python 3 is installed on your computer.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python main.py
```

> On some systems (macOS/Linux), you may need to use:
```bash
python3 main.py
```
